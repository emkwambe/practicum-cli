Q1: ETL stands for:
A) Extract, Transfer, Log
B) Extract, Transform, Load
C) Edit, Test, Launch
D) Evaluate, Train, Learn
Correct: B

Q2: An idempotent pipeline means:
A) It runs only once
B) Running it multiple times produces the same result without side effects
C) It's faster
D) It uses no memory
Correct: B

Q3: The best tool for the "Extract" stage from an API is:
A) awk
B) curl
C) sqlite3
D) sed
Correct: B

Q4: The best tool for the "Transform" stage of JSON data is:
A) grep
B) jq
C) curl
D) tail
Correct: B

Q5: The best tool for the "Load" stage (structured storage) is:
A) echo
B) sqlite3
C) cat
D) tee
Correct: B

Q6: Why save raw API responses before transforming?
A) APIs charge per request
B) If your transform logic has a bug, you can fix and reprocess without re-fetching
C) Raw is smaller
D) Required by HTTP spec
Correct: B

Q7: A production pipeline should include all of these EXCEPT:
A) Logging with timestamps
B) Error handling and retries
C) Hardcoded API keys in the script
D) Idempotent operations
Correct: C

Q8: cron is used in ETL pipelines for:
A) Data cleaning
B) Scheduling automated runs (e.g., daily at 6 AM)
C) JSON parsing
D) Database indexing
Correct: B

Q9: The join command requires input files to be:
A) In JSON format
B) Pre-sorted on the join column
C) The same length
D) In CSV format
Correct: B

Q10: paste -d',' file1.txt file2.txt does what?
A) Copies file1 into file2
B) Merges files side-by-side with comma as delimiter
C) Concatenates files vertically
D) Compares files
Correct: B

Q11: grep -rl "TODO" ./src/ | xargs wc -l does what?
A) Counts TODOs
B) Finds files containing "TODO" then counts lines in each of those files
C) Deletes TODO lines
D) Replaces TODO
Correct: B

Q12: diff <(sort file1) <(sort file2) uses what feature?
A) Here documents
B) Process substitution — feeds command output as temporary files to diff
C) Pipes
D) Redirection
Correct: B

Q13: xargs -P 4 means:
A) Limit to 4 arguments
B) Run up to 4 commands in parallel
C) Skip 4 lines
D) Print 4 times
Correct: B
