# Lesson: for Loops — Doing Something for Each Item

## [WHEN] You need this when:
1. You want to rename, copy, or process multiple files at once.
2. You need to run a command for each user in a list.
3. You are generating numbered output (1 through 100).
4. You want to iterate over command output (e.g., all .log files).

## [INTENT] "Do this for each item in the list."

## [CONTEXT]
A for loop repeats a block of commands once for each item in a list.
The list can be literal values, a file glob (*.txt), or command output.
The loop variable takes the value of each item in turn.

## [ACTION] for variable in list; do ... done
Common patterns:
  for file in *.txt; do               — all text files
      echo "Processing $file"
  done

  for i in 1 2 3 4 5; do              — literal list
      echo "Number $i"
  done

  for i in $(seq 1 10); do            — generated range
      echo "Count: $i"
  done

  for user in $(cut -d: -f1 /etc/passwd); do  — from command
      echo "User: $user"
  done

## [EXPECTED RESULT]
The commands inside the loop execute once per item.
After the last item, the loop ends and the script continues.

## Try it now:
  for color in red green blue; do echo "Color: $color"; done
  for f in /etc/*.conf; do echo "Config: $f"; done
