Q1: What does grep -v "DEBUG" server.log do?
A) Shows only lines containing DEBUG
B) Shows all lines EXCEPT those containing DEBUG
C) Deletes lines containing DEBUG from the file
D) Counts the number of DEBUG lines
Correct: B

Q2: How do you extract the first field from a colon-delimited file?
A) cut -f1 file.txt
B) cut -d: -f1 file.txt
C) grep -f1 file.txt
D) awk file.txt
Correct: B

Q3: Why must you sort data before using uniq?
A) uniq only works on sorted files
B) uniq removes only consecutive duplicates
C) sort makes uniq run faster
D) It is optional but recommended
Correct: B

Q4: What does wc -l count?
A) Words
B) Characters
C) Lines
D) Bytes
Correct: C

Q5: What does the pipe symbol | do?
A) Writes output to a file
B) Sends the output of one command as input to the next
C) Runs two commands at the same time
D) Compares the output of two commands
Correct: B

Q6: What pipeline finds the top 10 most frequent items?
A) sort | head -10
B) uniq -c | sort -rn | head -10
C) sort | uniq -c | sort -rn | head -10
D) grep -c | sort | head -10
Correct: C
Q7: What does sed -i 's/old/new/' file.txt do?
A) Prints the file with replacements but does not change it
B) Replaces "old" with "new" in the file permanently
C) Creates a new file called new.txt
D) Searches for "old" and deletes the line
Correct: B

Q8: How do you view the last 20 lines of a file?
A) head -20 file.txt
B) tail -20 file.txt
C) cat -20 file.txt
D) less -20 file.txt
Correct: B
