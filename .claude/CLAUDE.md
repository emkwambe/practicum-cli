# Practicum CLI

## Overview
CLI-native Linux courses teaching Intent → Context → Action → Result.

## Tech Stack
- Bash 4+
- Coreutils (no external dependencies)

## Philosophy
Every command shows why, what will happen, and what happened.

## Key Files
- practicum – main entrypoint
- lib/wrappers.sh – command sandboxing
- courses/ – lesson content
