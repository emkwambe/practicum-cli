Get Started

Install in 60 seconds.

Practicum CLI runs on any machine with bash. No root required. No Docker. No VM. Just your terminal.

🐧 Linux / WSL

Ubuntu, Debian, Fedora, Arch, etc.

Already have a terminal? One command:

curl -sL https://practicum-cli.dev/install.sh | bash

Or clone manually:

git clone https://github.com/emkwambe/practicum-cli.git
cd practicum-cli && chmod +x practicum
./practicum start

🪟 Windows (No Linux Experience)

Windows 10 / 11

Windows doesn't have a native Linux terminal, but WSL gives you a real one in minutes:

  1. Open PowerShell as Administrator
  2. Run: wsl --install
  3. Restart your computer when prompted
  4. Open "Ubuntu" from the Start menu
  5. Create a username and password
  6. You now have a real Linux terminal!

Then install Practicum CLI:

curl -sL https://practicum-cli.dev/install.sh | bash

🍎 macOS

Intel & Apple Silicon

macOS has a built-in terminal (Terminal.app or iTerm2). Open it and run:

curl -sL https://practicum-cli.dev/install.sh | bash

macOS uses zsh by default. Practicum CLI works on both bash and zsh.

☁️ Cloud / Chromebook

GitHub Codespaces, Gitpod, Crostini

No local machine needed. Use a free cloud terminal:

  1. Go to github.com/codespaces
  2. Create a new blank codespace
  3. In the terminal, run the install command
curl -sL https://practicum-cli.dev/install.sh | bash

Chromebook: Enable Linux (Settings → Advanced → Developers → Linux) then use the Terminal app.

🌱 Completely New to Linux?

Never opened a terminal before? Start here.

That's exactly who this course is for. Here's what you need to know:

  1. A terminal is a text-based interface to your computer. Instead of clicking icons, you type commands.
  2. Linux powers 90% of the world's servers, the cloud, Android phones, and most of the internet. Learning it opens doors to high-paying tech careers.
  3. You don't need to replace Windows or Mac. WSL (Windows) and Terminal.app (Mac) let you use Linux alongside your existing system.
  4. You can't break your computer. Practicum CLI has a safe sandbox mode where all commands are isolated. Practice freely.
  5. Start with Day 1. It teaches you the first 6 commands. By Day 3, you'll be writing scripts. By Day 10, you'll be job-ready.
